*******************************************************
Управление персонажем
*******************************************************
**W** - Движение вверх

**A** - Движение влево

**S** - Движение вниз

**D** - Движение вправо

**P** - Пауза

**F** - ???

*******************************************************
Режимы игры
*******************************************************
* **Обычный**
* **Countdown** (персонаж двигается по полю, пока не истечет время)
* **Inversed** (Режим движения персонажа в обратную сторону)
* **Сетевой**
